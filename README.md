# Consumo de servicio API REST - Clima

## Requerimientos mínimos

### Como producto se quiere:
Construir una SPA (Single Page Application) con JavaScript Vanilla para visualizar el clima de la ciudad de Concepción del Uruguay.

### Como Usuario:
Necesito poder visualizar una landing de contenido que me permita ver el clima
en el navegador.

--- 

## Criterios de Aceptación:
* La landing SPA debe de poder mostrar unas cards con las distintas variables del clima.
* La SPA deberá contener una Home Principal donde en forma de grid el Usuario las visualiza.

### Nombre de la Ciudad:
Para poder mostrar la información del clima como el resultado del json, se deberá utilizar la plataforma API REST de: https://openweathermap.org/
El desarrollador deberá contar con un API keys para poder acceder a la data de openweather..
